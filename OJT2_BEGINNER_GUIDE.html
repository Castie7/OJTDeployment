OJT2 Research System: Beginner Guide

Use this as the one-page checklist for daily operation and training.

Daily Start

  1. Open the system in the browser using the provided OJT2 link or shortcut.
  2. Log in using your assigned account.
  3. Use the left menu or dashboard cards to go to Workspace, Library, Masterlist, or Import.
  4. If a page does not load, refresh once before trying anything else.

If Server Connection Failed Appears

  1. Close extra browser tabs for the system.
  2. On the server computer, double-click the Start IIS shortcut.
  3. Wait until the restart window finishes.
  4. Open the system again or press refresh.
  5. If it still fails, call the IT/admin person.

Submit One Research Item

  1. Go to My Workspace.
  2. Click Submit New Item.
  3. Fill in required fields: Type, Title, and Author.
  4. Add optional details such as date, edition, publisher, ISBN/ISSN, shelf location, subjects, crop, and condition.
  5. Attach a PDF/image if available, or paste an external link in External Link.
  6. Click Submit Item.

New uploads start as private and pending. An admin can approve them and change visibility later.

Bulk CSV Import

  1. Go to Import.
  2. Click Download Template.
  3. Fill the spreadsheet. Keep at least Title and Author.
  4. For online-only materials, use the Link column. Accepted aliases are URL, External Link, or Website.
  5. Save as CSV UTF-8.
  6. Upload the CSV, review the preview table, then click Confirm & Import.

Bulk PDF Upload

  1. Go to Import, then use Bulk PDF Upload.
  2. Name each PDF close to the research title.
  3. If there are multiple editions, add a hint in the file name, such as Golden Roots [Vol. 1].pdf.
  4. Review the match preview before upload.
  5. If a PDF does not match automatically, use Manual link to empty record.
  6. If unsure, leave it not linked and ask the admin before uploading.

Viewing Research

  1. Click a research title from the Library, Workspace, or Masterlist.
  2. Read the catalog details: author, edition, publisher, date, subjects, shelf location, and condition.
  3. In Digital Access, open the PDF preview if a PDF exists.
  4. If an external link exists, click Open External Link / Website.
  5. If no PDF or link exists, the item is metadata-only.

Warnings And What To Do

Message Meaning Action
Similar title found A close title already exists. Review the existing title. Continue only if it is truly a different item.
Duplicate The same title/author or record already exists. Do not create another copy. Check if it only needs a PDF/link attached.
No PDF linked The record exists but has no attached PDF. Attach the correct PDF later, or add an external link if available.
Server Connection Failed The app cannot reach the server. Use the Start IIS shortcut, refresh, then call IT/admin if it continues.

Before Finishing The Day

  • Check that important uploads are visible in Workspace or Masterlist.
  • Make sure pending items are not accidentally left public.
  • Review duplicate or similar-title warnings before approving records.
  • Log out when done.

Tip for trainers: teach one workflow at a time. Start with one manual upload, then one CSV import, then one PDF attach.
